# Condaty - Web

Este proyecto es la landing page para **Condaty**, desarrollada con **Next.js**, **TypeScript** y **Tailwind CSS**.

## Características

- **SEO Optimizado**: Uso de Next.js App Router, Metadata API, Sitemap y Robots.txt.
- **Diseño Moderno**: Estilizado con Tailwind CSS.
- **Iconos**: Utiliza `lucide-react`.
- **Animaciones**: Preparado para usar `framer-motion`.

## Requisitos

- Node.js 18+
- npm o yarn

## Instalación

```bash
npm install
```

## Desarrollo

Para iniciar el servidor de desarrollo:

```bash
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000) en tu navegador.

## Construcción

Para generar la versión de producción:

```bash
npm run build
npm start
```

## Estructura del Proyecto

- `src/app`: Contiene las páginas y el layout principal (App Router).
- `src/components`: (Opcional) Para componentes reutilizables.
- `public`: Archivos estáticos.
